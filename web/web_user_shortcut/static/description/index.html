Odoo shortcut to frequently used menu items

Quick add/remove shortcut button

Set shortcut's name as individual prefer

Toggle show shortcuts

Advance settings

This module brings some features:

  • Quick add shortcut from current work menu item
  • Quick remove shortcut from current work menu item
  • Show systray icon to toggle list of shortcut (only if user added any shortcut)
  • Show list of shortcuts, "jump" to clicked shortcut
